# Foresight Map — Data Collection Brief

**What I need:** Look through our existing data (wherever it lives — spreadsheets, CRM, past lists) and pull together one clean dataset for **all** past and current Foresight fellows, grantees, and prize winners so we can make the map live. We’re not asking you to contact everyone; just find and compile what we already have (and note what’s missing).

**Same fields for everyone** (alumni and current):

- **Full name** (as they want it on the map)
- **Role:** Fellow | Grantee | Prize Winner
- **Cohort start year** (year they joined)
- **End year** (for alumni only; last year active — if one year only, same as start; leave blank for current)
- **Alumni or current?** (Alumni = no longer active; Current = still active)
- **Location:** city + country (one location per person — where they are based now or last known)
- **Focus areas** (at least one): Secure AI | Neurotechnology | Longevity Biotechnology | Nanotechnology | Space | Existential Hope | Other
- **Short project tagline** (one sentence)
- **Expanded project description** (optional, a few sentences)
- **Affiliation / institution** (optional — university, company, or org)
- **Profile URL** (optional)
- **Contact** (email or public handle — we want this for everybody when possible)

**Primary node** (current only): If they’re tied to a node, use Global | Berlin Node | Bay Area Node. Alumni don’t have a node.

**Deliver:** One spreadsheet — **People** (one row per person, all fields above). Use the exact values for role and focus areas so we can load the data without cleanup.
